# JADY 💎
Your diary and journal *backwards*

## About
JADY or Jady is a lightning fast and very easy to use diary application for Windows and Linux. Since JADY is built with <a href="https://avaloniaui.net/">Avalonia</a>, the app renders its own UI rather than relying on the operating system's native buttons and menus, which translates into a smooth experience no matter the platform you are on.
